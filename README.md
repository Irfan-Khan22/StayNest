# StayNest

StayNest is a full-stack web application inspired by Airbnb that allows users to browse, create, edit, and manage property listings. Users can also register, log in, upload listing images, and write reviews.

## Live Demo

[https://your-render-url.onrender.com](https://staynest-brw6.onrender.com)

## Features

- User Authentication (Signup, Login, Logout)
- Create, Edit, and Delete Listings
- Upload Listing Images
- Add and Delete Reviews
- Session-based Authentication
- Flash Messages
- Responsive User Interface

## Installation

```bash
git clone https://github.com/Irfan-Khan22/StayNest.git

cd StayNest

npm install

npm start
```

## Environment Variables

Create a `.env` file and add:

```env
ATLASDB_URL=
CLOUD_NAME=
CLOUD_API_KEY=
CLOUD_API_SECRET=
MAP_TOKEN=
SECRET=
```
